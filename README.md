# colinkiiim.github.io

这个目录是 `日计-DIY` 的独立 GitHub Pages 文章站本地骨架。

## 目标地址

- 用户站根地址：`https://colinkiiim.github.io/`
- 文章列表：`https://colinkiiim.github.io/post/`
- GitHub Token 教程：`https://colinkiiim.github.io/post/github-token-guide/`

## 为什么是这个仓库名

GitHub Pages 用户站需要使用：

- `用户名.github.io`

你的 GitHub 用户名是 `ColinKiiim`，发布域名会按小写生效，所以这里使用：

- `colinkiiim.github.io`

## 目录结构

- `index.html`
- `post/index.html`
- `post/github-token-guide/index.html`
- `assets/site.css`
- `.nojekyll`

## 后续怎么发

1. 在 GitHub 新建仓库 `colinkiiim.github.io`
2. 把这个目录初始化成独立 git 仓库
3. 推送到 GitHub
4. 在 GitHub Pages 里确认从默认分支根目录发布

推送完成后，App 里的 `GitHub Token 教程` 链接就可以从当前站内路径切到 GitHub Pages 文章地址
